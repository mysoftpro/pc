Слева компоненты wit, которые просто выгребаются по каталогам. Компоненты в которых есть map_to_new перебрасываются в центр. Компоненты с 'new_' выкидываются. Справа зеденым цветом отмечены компоненты wit у которых есть catalogs. Т.е. это самостоятельные компоненты и у них не может быть маппинга. Если в центре появится new компонент синего цвета - значит у него есть catalogs! (это самостоятельный компонент!) Работа выгдядит следующим образом. Делается поиск по компоненту слева. Если есть результат - делается маппинг. Если нет - элемент слева закрывается. И так до тех пор, пока слева не останется ничего. Справа должны остаться только зеленые штуки. Если остались черные - нужно создавать компоненты new на другой странице.
